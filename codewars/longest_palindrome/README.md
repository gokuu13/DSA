## Longest Palindrome

Find the length of the longest substring in the given string s that is a  
palindrome.  

### Example:
    "I like racecars that go fast" => 8 ("racecar")
    "a" -> 1 
    "aab" -> 2  
    "abcde" -> 1
    "zzbaabcd" -> 4
    "" -> 0
    
### Note:
If length of input is 0, the return value  must be 0.
