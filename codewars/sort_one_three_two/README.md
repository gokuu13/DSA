## Sort - one, three, two

### Task:
Create a function to sort a vector of integers by name.

### Notes:
* $arr[i] \in [0, 999]$
* array may have duplicates
* array may be empty

### Example:
    sort({1, 2, 3, 4}) => {4, 1, 3, 2}

    named array => {"one", "two", "three", "four"}
    
    sorted by name =>{"four", "one", "three", "two"}
